# Selody-Project

## 프로젝트 소개

개인 일정을 기반으로 한 그룹 일정 관리 커뮤니티

## 주요 기능

- 개인 일정 공유 기능:
    - 개인 일정을 등록하고 해당 일정을 같은 그룹원에 공유 가능합니다.
- 그룹 공유 일정 기능:
    - 개인 일정들을 바탕으로, 그룹간의 공유 일정을 정하여 약속을 쉽게 잡을 수 있도록 도와줍니다.
- 커뮤니티:
    - 그룹별 피드를 통해 그룹원 간에 소통이 가능합니다. 

## 기술 스택

- Front-end:
  - React
  - Redux-Toolkit
  - React-Router v6.4
  - JavaScript
- Back-end:
  - Node.js
  - Express
  - mySQL
  - sequelize
  - REST-API
  - Docker
  - jest
